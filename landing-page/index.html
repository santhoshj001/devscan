How it gets the job done

01

Rapid Scan

Run the CLI. DevScan recursively searches your directories for 15+ different project types, intelligently bypassing nested dependencies.

02

Analyze & Classify

Checks filesystem metadata and shells out to Git to evaluate project health, generating a local JSON report instantly.

03

Review & Clean

A beautiful local web dashboard launches allowing you to review suggestions, push git work, or move stale projects to the macOS Trash.
